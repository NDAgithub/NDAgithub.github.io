Noah Davis-Abraham

Hi! I'm Noah, and in 2023 I graduated from Memorial University of Newfoundland and Labrador's Computer Science program, specializing in Visual Computing and Games.

I’ve always wanted to create video games, since I grew up playing and sharing them with my friends and family. Eventually I want to make games that create the same feelings I felt while playing Omori, Firewatch, Eastward, Return of the Obra Dinn and Undertale. Recently, myself and a team of 8 friends have put our dreams into motion and have started developing our first RPG together in Unity.

For the last decade I have done improvisational theatre and comedy, which has helped me to grow as both a good communicator and a compelling storyteller. My experience has helped me understand games from player and programmer perspectives, and I’m excited to bring this knowledge into my future projects!

Have You Seen Kent?

COMP4300 - Introduction to Game Programming Final Project

Have You Seen Kent? is a game that was made using only C++ and the SFML library for the final project of Memorial University's COMP4300 class. Myself, Andrew Harris, Collin Riggs and Justin Butler worked on the game together to build the systems for physics (collision, ray casting, overlap), AI pathfinding, inventories, shops, item usage, saving and loading game states and rooms, options, lighting effects, and an in-game level editor. GDMC Competition 2022 Submission

COMP4303 - Artificial Intelligence in Computer Games Final Project

The Generative Design in Minecraft (GDMC) Competition is an annual competition in which participants create programs to generate unique Minecraft settlements. For this project, I created a Python program that used an HTTP server mod to introduce ocean and desert villages to the game. I used a 2D representation of the world’s surface to determine reasonable house locations before building houses modularly, with their sizes based on their surrounding area. Houses were connected to nearby houses with BFS pathfinding. A trailer and video essay are viewable below.

Yuwanna Fight

COMP3100 - Web Programming Final Project

Yuwanna Fight is a game developed by myself and Collin Riggs as our final project for Memorial University’s Web Programming course. We utilized EJS to supplement HTML code while building the screens of our game, and used a RESTful API alongside node.js modules to transfer the game’s information between files. A user database was implemented using MongoDB, and users were registered using a custom authentication system. A video presentation demonstrating the results is viewable below.

Hand-Recognition Program

COMP4301 - Computer Vision Final Project

For our final project in Memorial’s Computer Vision course, Collin Riggs, Justin Butler and myself created a Python program that uses a neural network to recognize hand gestures from a camera input to perform system commands, such as playing/pausing media. OpenCV, MediaPipe, TensorFlow, scikit-learn and other ibraries were used in the creation of this project, alongside experience from implementing image manipulation in this course and Memorial’s Visual Computing and Applications course.
